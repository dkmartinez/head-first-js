How Properties work

How to access a property. To access a rpoperty in an object, start with the object name, follow it with a period (dot) and then use the property name.

How to change a property. You can change the value of a property at any time. All you need to do is assign the property to a new value.

You can use an object's property just like you use a variable, except you need to use dot (.) notation to access the property in the object.
